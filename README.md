# Drift
In-game text chat overlay for games that do not have text chat

## Overview
Drift is an Electron-based chat overlay that sits on top of games and applications, providing real-time text communication where native chat isn't available. Perfect for voice-only games or situations where you need accessible text chat.

## Features (Current)
- Transparent, always-on-top overlay
- Basic text chat interface
- Auto-scroll to latest messages
- Minimal, frameless UI
