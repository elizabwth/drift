# Wishlist
Exploratory and low-priority feature ideas for future consideration

## Voiceline System
A complete voiceline/soundboard system for quick audio communication

- [ ] Voicewheel or voiceline menu UI
  - [ ] Grid or radial menu for quick access
  - [ ] Visual preview of available voicelines
  - [ ] Customizable layout
- [ ] Audio playback
  - [ ] Play audio clips through overlay
  - [ ] Volume control and mixing options
  - [ ] Audio output device selection
- [ ] Voiceline management
  - [ ] Import custom voiceline packs
  - [ ] Create and organize custom soundboards
  - [ ] Share voiceline packs with other users
- [ ] Accessibility
  - [ ] Subtitle/caption support for voicelines
  - [ ] Hotkey and controller support for triggering voicelines
  - [ ] Text-to-speech integration for custom messages
